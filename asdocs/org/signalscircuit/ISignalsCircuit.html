loadComponent()method 
public function loadComponent(uri:String, success:Signal = null, failure:Signal = null):void

This method is implemented to provide a mechanism for loading a component, typically from a remote URL location. Before sending the success Signal this method makes sure to run the registerComponent() method for you. Loading the SWF and calling the registerComponent() method can be done manually, this method just does it for you and dispatches the Signal object that is passed in as the success Signal or the failure Signal if something fails.

Parameters

uri:String — A unique resource identifier for the component. Typically a relative path to a ISignalsCircuitComponent.swf.
 
success:Signal (default = null) — A Signal to dispatch if the component is loaded successfully. The success Signal should expect a Loader object.
 
failure:Signal (default = null) — A Signal to dispatch if the component load fails. The failure Signal should expect an Event object from the failure.

registerApplicationToComponentSignal()method 
public function registerApplicationToComponentSignal(signalName:String, signal:Signal):void

Signals that need to be sent to components by the application can be registered with the registerApplicationToComponentSignal() method. Whenever this Signal is dispatched its name and arguments will be sent to all components. If the component has registered a Signal with SignalsCircuit's registerApplicationSignalNameToReceive() the component can react to a Signal defined by the component that expects the same arguments as the application sent. Objects mapped in the arguments array are encoded/decoded using the AMF protocol to go from one application sandbox to the other.

Parameters

signalName:String
 
signal:Signal

registerComponent()method 
public function registerComponent(uri:String, component:Loader):void

When a ISignalsCircuitApplication has finished loading a ISignalsCircuitComponent object it should use this method to register the component as an ISignalsCircuitComponent to be able to send and receive signals from it. The Loader object that loaded the component is passed into the method to register it.

Parameters

uri:String — A unique resource identifier for the component. Typically a relative path to a ISignalsCircuitComponent.swf.
 
component:Loader — A Loader object that has loaded into its content property a SWF that implements ISignalsCircuitComponent
