Dispatches a Signal in a application that has registered the signalName with the SignalsCircuit registerComponentSignalNameToReceive() method. This method is used by SignalsCircuit to send a message to the parent application that loaded it if a component has registered a Signal using the SignalsCircuit registerComponentToApplicationSignal() method.

receiveSignalFromApplication()method 
public function receiveSignalFromApplication(signalName:String, arguments:String):void

This method is used by SignalsCircuit when a Signal is dispatched that has been registered with SignalsCircuit using its registerApplicationToComponentSignal() method to trigger dispatching a Signal by its registered name in all loaded components.

Parameters

signalName:String — The name of the Signal that the application is sending out.
 
arguments:String — An array of arguments to dispatch with the Signal if the component has a Signal for that name registered, encoded as an AMF base64 encoded String.
